#  CitySearch

Application that makes searching for the population data of cities easier. Built using Kotlin in React Native with OkHttp and Gson.

## Setting up the project environment

This section will cover how to set up the project environment for Android studio using Git and Gradle

#### Git

If you do not have Git installed on the command line you can manually download the project or set up Git in command line using the following documentation https://docs.gitlab.com/ee/gitlab-basics/start-using-git.html

If you have Git installed on the command line run the following commands to clone the repository onto your computer

    cd directory
    git init
    git clone https://github.com/manngard/CitySearch

#### Android Studio and Gradle

To set up the project through Android studio the latest version of the program can be downloaded at https://developer.android.com/studio

Import the project as an external Gradle module and run the build.gradle file. This will synchronize and download all necessary dependencies making the application ready to run
